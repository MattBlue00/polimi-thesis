I can help you with data imputation. Here's how we can approach this:

1. For numerical columns (price, bed, bath, acre_lot, house_size):
- Replace missing values with the median or mean
- Remove invalid values (like 9999 in 'bed')
- Convert price strings to numerical values

2. For categorical columns (brokered_by, status, street, city, state, zip_code):
- Replace missing values with the mode
- Standardize formats (e.g., state abbreviations)

Here's a Python script to perform the imputation:

```python
import pandas as pd
import numpy as np

# Convert the data to a DataFrame
df = pd.DataFrame(data)

# Clean price column
df['price'] = df['price'].replace('-', np.nan)
df['price'] = df['price'].str.replace('$', '').str.replace(',', '')
df['price'] = pd.to_numeric(df['price'])

# Clean bed column
df['bed'] = df['bed'].replace(9999, np.nan)
df['bed'] = pd.to_numeric(df['bed'])

# Convert date
df['prev_sold_date'] = pd.to_datetime(df['prev_sold_date'], errors='coerce')

# Impute numerical columns with median
numerical_columns = ['price', 'bed', 'bath', 'acre_lot', 'house_size']
for col in numerical_columns:
    df[col] = pd.to_numeric(df[col], errors='coerce')
    median_value = df[col].median()
    df[col] = df[col].fillna(median_value)

# Impute categorical columns with mode
categorical_columns = ['brokered_by', 'status', 'street', 'city', 'state', 'zip_code']
for col in categorical_columns:
    mode_value = df[col].mode()[0]
    df[col] = df[col].fillna(mode_value)

# Standardize state names
df['state'] = df['state'].replace({'TX': 'Texas', 'NC': 'North Carolina', 'SC': 'South Carolina',
                                  'OK': 'Oklahoma', 'MN': 'Minnesota', 'PA': 'Pennsylvania'})

# Clean status values
df['status'] = df['status'].replace({'s': 'sold', 'f': 'for_sale'})
```

This script will:
1. Clean the price column by removing '$' and ',' and converting to numeric
2. Remove invalid bed values (9999)
3. Convert dates to datetime format
4. Impute numerical missing values with median
5. Impute categorical missing values with mode
6. Standardize state abbreviations
7. Clean status values
